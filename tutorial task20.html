Book Title

Web Technologies: Black Book

Author

Kogent Learning Solutions Inc.

Publisher

Dreamtech Press

Category

Web Development and Programming

Summary

Web Technologies: Black Book is a comprehensive guide for learning web development technologies. The book covers HTML, CSS, JavaScript, XML, PHP, and other web programming concepts. It provides practical examples and applications that help students understand website design and development. The book is widely used by engineering students and web developers as a reference for modern web technologies.
