Database Management Systems

Course Code: DBMS201

Learn database concepts, data modeling, SQL, and normalization.

Learning Outcomes

  • Understand database architecture.
  • Write SQL queries for data manipulation.
  • Design normalized database schemas.

Learning Outcomes
